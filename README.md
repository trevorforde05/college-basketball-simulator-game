# College Life Simulator - Interactive Text-Based Game

This project is a branching narrative simulation game built using Twine that models decision-based outcomes in a college athlete's academic and athletic career.

Players make strategic choices between studying and basketball practice, which dynamically affect GPA and basketball performance metrics. The system tracks user decisions and adjusts gameplay outcomes based on performance trade-offs, creating multiple gameplay paths and replayable experiences.

## Features
- Choice-based decision mechanics
- GPA performance tracking
- Basketball stat tracking
- Outcome-based gameplay paths
- Interactive narrative system

## Built With
- Twine
- HTML
